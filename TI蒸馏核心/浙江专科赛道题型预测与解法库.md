## 浙江专科赛道判断

浙江奇数年赛区通常使用全国题;偶数年更可能是独立省赛或与专题赛结合。对专科/高职队伍来说,重点不是押最难题,而是准备“工程闭环短、模块明确、测试可重复”的题。

优先级:

1. 小车/运动控制/任务执行
2. 信号测量/简易仪器
3. 声光/超声/非接触定位测量
4. 视觉识别/摄像测量
5. LED/交互/控制盘/综合装置
6. 简化电源/功率测量/模拟模块

不建议把主力押在高压大功率电源、复杂三相电能变换、纯射频高频链路。它们上限高,但专科队伍若硬件和仪器储备不足,调试风险大。

- 2021 I 具有发电功能的储能小车

- 2023 I 气垫悬浮车

这些题的共同特征:

- 更强调“装置能稳定完成任务”,不一定要求复杂理论推导。
- 常见输入是传感器、按键、声光、线路、距离、位置。
- 常见输出是运动执行、LED/显示、人机交互、报警、定位结果。
- 成败通常取决于机械/传感器安装、抗干扰、状态机、调试流程。

## 最可能方向一: 自动小车/移动执行装置

可能题面:

- 自动避障小车、循迹小车、储能/发电小车、坡道行驶、跟随、定点停车、搬运/送货、场地任务。

为什么高概率:

- 2021 I、2025 K 是高职高专核心方向;浙江省赛和校赛也常用小车题筛选队伍。
- 器材易统一,评测直观,适合专科工程实践。

最佳基础方案:

- MCU: MSPM0 / STM32F103/F401 / ESP32 只做通信或辅助。
- 传感: 编码器 + 灰度/红外循迹 + 超声/ToF 避障 + IMU 可选。
- 执行: 双路直流减速电机 + TB6612/DRV8833/电调 + 独立电源。
- 算法: 状态机 + 速度闭环 PID + 转向规则/PID + 任务节点表。

赛题发布后优先问:

- 场地尺寸、路线颜色/材质、障碍形状、是否允许标志物、计时方式、停车误差、是否要求自动启动。

拿分策略:

- 第一天完成底盘闭环和基础路线。
- 第二天完成任务流程和异常恢复。
- 第三天优化速度、停车精度、发挥项。

## 最可能方向二: 简易仪器/信号测量

可能题面:

- 周期信号识别、频率/幅值/相位测量、线路故障检测、电缆测试、功率/照度测量、RLC/电路模型测量。

为什么高概率:

- 2021 J、2023 J、2025 D/G 是近年稳定方向。
- 专科赛道可通过“仪器化装置”考查电路、嵌入式、显示和校准。

最佳基础方案:

- 模拟前端: 分压/保护/滤波/运放缓冲/整形比较。
- 采样: ADC + DMA 或定时器捕获。
- 算法: RMS、峰峰值、过零频率、FFT/Goertzel、简单模型拟合。
- 输出: OLED/TFT + 按键菜单 + 校准参数保存。

赛题发布后优先问:

- 被测信号范围、频率范围、精度要求、输入阻抗、是否隔离、测试时间、标准仪器对比方法。

拿分策略:

- 先做单一量程准确,再自动量程。
- 先做基本参数,再做识别/诊断。
- 必须准备校准表,不要裸算。

## 最可能方向三: 声光/超声定位与非接触控制

可能题面:

- 超声信标定位、声音定位、非接触控制盘、距离/角度测量、光照稳定控制。

为什么高概率:

- 2021 K、2025 I、2025 J 是专科友好的传感器综合题。
- 硬件成本低,评测可量化。

最佳基础方案:

- 超声/声定位: 多接收头 + 同步计时/采样 + 阈值触发 + 几何解算。
- 非接触控制: 电容触摸/红外/ToF/霍尔/视觉,优先选择抗干扰最强者。
- 光照控制: 光敏/照度传感器 + PWM 调光 + PID/分段控制。

赛题发布后优先问:

- 测距范围、定位区域、目标数量、环境噪声、允许标定时间、遮挡和反射条件。

避坑:

- 超声题难点不是公式,是同步、回波、多径和阈值。
- 非接触题难点是误触和环境变化。
- 光照题要锁定传感器位置,否则闭环会漂。

## 最可能方向四: 视觉识别和简单智能装置

可能题面:

- 棋盘/图案识别、目标测距、物体巡查、自动瞄准、二维码/颜色识别、计数装置。

为什么中高概率:

- 2024 E、2025 C/E/H 等题体现“传感器 + 识别 + 控制”趋势。
- 但专科赛道通常不会要求复杂深度学习,更可能是规则视觉或固定场景识别。

最佳基础方案:

- OpenMV / ESP32-S3 / 树莓派 / 核桃派做视觉。
- 下位机 MCU 做电机、舵机、按键、显示。
- 算法优先级: 颜色阈值 > 边缘/轮廓 > 模板/AprilTag > 轻量模型。

赛题发布后优先问:

- 光照是否固定、背景是否统一、目标尺寸/颜色、相机安装位置、是否允许标记。

避坑:

- 不要一开始训练模型。先用固定光照和规则视觉拿基本分。
- 视觉输出不要直接控制电机,要给下位机目标坐标/偏差。

## 最可能方向五: LED/交互/显示控制装置

可能题面:

- LED 显示、控制盘、辨音识键、台灯、游戏装置、人机交互。

为什么中等概率:

- 2019 I、2021 K、2023 K、2025 I 都是专科可做的综合装置。

最佳基础方案:

- MCU + LED/数码管/OLED/TFT + 按键/旋钮/触摸/传感器。
- 软件重点是状态机、刷新、去抖、参数保存。
- 若涉及音频识别,优先用频率/能量/节拍特征,不要上复杂语音识别。

## 赛前必须准备的通用模块

硬件:

- 稳定 5V/3.3V 电源板,带保险/反接保护。
- MSPM0 或 STM32 最小系统 + 下载调试器。
- 电机驱动板 + 编码器电机 + 标准小车底盘。
- OLED/TFT 显示 + 按键/旋钮/蜂鸣器。
- 常用传感器: 灰度、红外、超声、ToF、IMU、光敏/照度、电容触摸。
- 模拟前端小板: 分压、保护、运放缓冲、比较器、滤波。

软件:

- 状态机模板。
- PID 速度环/位置环模板。
- ADC+DMA 采样模板。
- 定时器输入捕获模板。
- OLED/TFT 菜单模板。
- 串口调试协议。
- 数据校准表和参数保存。

调试:

- 每个模块必须有单独测试程序。
- 比赛当天不要临时写底层驱动。
- 所有传感器必须有串口输出原始值的调试模式。

## 赛题发来后的我的决策规则

我会按下面顺序给方案:

1. 判断是否适合专科队伍做。
2. 给“稳拿基本分方案”,优先保证能交付。
3. 给“冲奖发挥分方案”,但标注风险。
4. 给“不要做/晚点做”的高风险项。
5. 给 72 小时任务拆解。
6. 给每个模块的验收测试。
7. 对照本地历年题和开源作品,找最近似参考。

## 最终押题结论

如果只准备 4 个方向,优先级如下:

1. 自动小车/移动执行装置
2. 简易仪器/信号测量
3. 超声/非接触/声光定位
4. 视觉识别 + 简单控制

如果再加一个方向,准备 LED/交互/显示控制。

电源/功率电子可以准备基础知识和安全测量,但专科队伍不建议把它作为唯一主攻,除非队伍有成熟电源硬件经验和足够仪器。
