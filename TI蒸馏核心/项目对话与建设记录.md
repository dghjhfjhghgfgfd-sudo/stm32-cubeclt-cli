## 七、关键技术发现(可复用)

- 浙江省赛官网是 SPA,正文须走逆向 API `cc.moocollege.com/web/competition/news(/detail)`,而非直接抓 HTML。
- moocollege OSS 附件 **2024-07 为时间归档分界**,更早一律 403。
- **RAR5 解压无需 WinRAR**:`C:\WINDOWS\system32\tar.exe -xf xxx.rar` 即可(本环境实测成功)。
- 大仓库 `Invoke-WebRequest` 易挂死,改用 `curl.exe -L -f --max-time` 下载 + `tar.exe` 解压(解决深层中文长路径 `Expand-Archive` 失败)。
- PowerShell 5.1 把无 BOM 的 UTF-8 `.ps1` 当 GBK 读,脚本体中文字面量会乱码崩溃 → 脚本逻辑保持 ASCII 或用 Write 工具写文件。
- Workflow 脚本须是纯 JavaScript,TypeScript 类型注解会解析失败。
