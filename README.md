## 使用方式

比赛题目发布后，先根据赛题类型匹配：

1. 小车/控制/执行机构
2. 测量/信号分析/仪器
3. 视觉/自瞄/坐标识别
4. 声源/超声/非接触定位
5. 电源/功率类高风险题

然后从方案雷达里选择主控、模块、可复用工程和第一天可交付目标。
